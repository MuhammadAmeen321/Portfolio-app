# Auto Attendance System Image Recognition

## Project Type
Computer Vision and Deep Learning

## Problem Statement
Manual attendance tracking is time-consuming and prone to human error.

## Solution
An automatic attendance system based on face verification using a Siamese Network.

## Core Features
- Face image preprocessing
- Siamese embedding comparison
- Match threshold for identity verification
- Attendance mark generation and logging

## Tech Stack
Python, TensorFlow, Siamese Network, OpenCV

## Outcome
The system automates attendance with better reliability and reduces manual administrative work.
