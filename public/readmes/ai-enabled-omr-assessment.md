# AI enabled E assessment of close ended questions on OMR sheet

## Project Type
Final Year Project (FYP)

## Problem Statement
Manual checking of OMR answer sheets is slow and can introduce checking inconsistencies.

## Solution
An AI-enabled e-assessment system that reads close-ended OMR responses and evaluates them automatically.

## Core Features
- OMR sheet detection and alignment
- Bubble recognition and answer extraction
- Automatic score calculation
- Result summary generation

## Tech Stack
Python, OpenCV, Machine Learning, Flask

## Outcome
The project reduces checking time and provides more consistent assessment for objective exams.
