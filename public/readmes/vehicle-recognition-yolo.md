# Vehicle Recognition System Using YOLO

## Project Type
Deep Learning Project

## Problem Statement
Vehicle detection is needed for smart traffic analysis and surveillance systems.

## Solution
A vehicle recognition system built with YOLO for real-time object detection.

## Dataset
Kaggle vehicle dataset used for training and validation.

## Core Features
- Vehicle detection from images/video frames
- Bounding box and class predictions
- Real-time inference support

## Tech Stack
Python, YOLO, PyTorch, OpenCV, Kaggle Dataset

## Outcome
The model provides accurate and fast vehicle recognition suitable for practical CV applications.
